# slu-alumni-engagement-portal
Final project for IS-5960: Alumni Engagement Portal
Overview
The SLU Alumni Engagement Portal is a web-based platform designed to centralize alumni activities including events, mentorship, donations, and feedback into one unified system. It helps SLU track engagement patterns and improve the way alumni interact with the university.
Features
Profile Management – Alumni can view or update their basic details.
Events Module – View events, see attendance, and explore engagement trends.
Mentorship Matching – Request mentorship or volunteer as a mentor.
Donations Dashboard – Track giving patterns and donor activity.
Feedback Tool – Submit feedback and view participation by graduation year.
Analytics Dashboards – Interactive charts built with HTML, CSS, JavaScript, and CSV data.
 Technologies Used
Frontend: HTML, CSS, JavaScript
•	Data Storage: CSV files + browser Local Storage
•	Synthetic Data: Generated using python, the faker library
•	Platform: Google Sites (for project presentation)
 Ethical & Social Considerations 
•	Data Privacy: Only non-sensitive, synthetic data is used.
•	Inclusion & Fairness: All alumni features are equally accessible to everyone.
•	Bias Reduction: Dashboards show trends not individual-level comparisons.
•	Accessibility: Works across devices, ensuring broad access for alumni.
•	Sustainability: Consolidates multiple functions into one system to reduce digital clutter.
